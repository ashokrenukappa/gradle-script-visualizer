How to install and run this application:
  • Download & Install GraphViz. This is what actually generates the graphics (my program just generates the data files in a format that GraphViz understands).
  • Associate .dot files with GraphViz
    • This lets the app display the graphics by simply telling the file system to open them, rather than calling GraphViz directly
  • Either:
    • Run via Java WebStart
      • Accept all dialogs.
      • If it asks to load a newer version of Java or WebStart, say "yes".
      • Ignore the warnings about installing software on your system or accessing your file system. I'm using the Java WebStart architecture, and the cost of that is annoying dialogs like this for the users. But it is nice enough to ask before touching your file system, so you know what's going on.
      • Launch it twice from this web page and it'll install icons on to your desktop and start menu. You won't have to visit this site again to run the application after that.
      • You'll always have the most up-to-date version - it updates to new versions automatically! This is especially nice when bugs are fixed or features are added.
      • You don't need to be connected to the Internet (or this server) to use the application, but if you are connected, it will update itself if necessary.
    • Or, download the zip file and run from script
      • Both Unix and DOS scripts are included in the bin directory
      • The drawback is you have to download every time there's an update, it's not automatic like WebStart
